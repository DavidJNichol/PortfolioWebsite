Navmesh Agents in Unity
Tags:
C#
Unity
Navmesh
Proximity Detection Behaviors
Solo
February 2020
3D C# agent AI stealth demo using Unity's Navmesh system. Based on a decision tree that encompasses interactions triggered by proximity and sound. Agent behaviors include seeing, smelling and hearing. Hearing is done with a large box collision radius, but has a lower chance of detection than hearing, where detection is based on a random chance of alerting the agent every second the player is colliding with that box. Smelling detection has a shorter radius but larger chance of detection. Sight is a raycast from the eyes of the agent and has a 100% chance of alerting the agent. Once the agents have been in chasing mode for too long, they go into a resting mode, where they run to a certain area to replenish their energy.
